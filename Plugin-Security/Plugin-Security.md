### 插件安全

恭喜，你的代码可以工作了！但是它安全吗？如果他们的网站被黑了你的插件可以保护你的用户的信息吗？在WordPress.org中最好的插件可以保护他们用户的信息安全

请记住(Please keep in mind) 你的代码或许可以运行在数百甚至上百万的网上，但是安全是最最重要的

本章我们将讲解如何检查用户，包括验证以及清洗用户输入内容和输出内容，创建验证问题
